# Photos Folder

Add your photos here! 

## For the first page (index.html):
- `photo1.jpg` - First memory photo
- `photo2.jpg` - Second memory photo  
- `photo3.jpg` - Third memory photo

## For the surprise page (surprise.html):
- `surprise1.jpg` - Special memory photo 1
- `surprise2.jpg` - Special memory photo 2
- `surprise3.png` - Special memory photo 3
